Was sind denn die wichtigsten Fragen?

Woran erkennt man die wichtigsten Fragen

Man kann ja nicht alles im Detail klären.

Beispiel: Was kann eine Architektur-Entscheidung sein?

Aufgabe: Nummeriere die Zeilen eines Listings mit Zeilennummern im Browser

Entscheidung: CSS-Counter können das (https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Lists_and_Counters/Using_CSS_counters), also wird das mit CSS gemacht

DAS ist Architektur?

Neue Anforderung: Man soll am code-Element angeben können, mit welcher Zeilennummer die Nummerierung beginnt (für Fortsetzungen)

CSS kann so etwas aus einem Daten-Attribut holen: https://developer.mozilla.org/de/docs/Web/Guide/HTML/Using_data_attributes

Leider kann man das nicht für Counter nehmen, weil die Numbers brauchen, die Konvertierung von String-Attribut mach Number ist zwar spezifiziert, aber wird von keinem Browser unterstützt: https://developer.mozilla.org/en/docs/Web/CSS/attr

Hier geht es also nicht weiter, das war also eine wichtige Entscheidung ()
